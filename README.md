# llm_utils

Utilities and examples for training/inference of language models
using HuggingFace and PyTorch Lightning

## Structure of this repository

[huggingface](./huggingface) contains utilities for using the HuggingFace libraries.

[examples](./examples) contains various examples for processing data,
training models, evaluating models, etc. Just miscellaneous stuff
that wasn't significant enough to dedicate to an entire project/repo.
